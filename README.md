# -# منصه خرصان للصرافه

منصة لإدارة رسائل العملاء وربطها بتطبيق Auto Forward SMS. هذه المنصة تتيح لك:

- استقبال جميع رسائل العملاء من الجوال مباشرة.
- عرض الرسائل للموظفين في لوحة التحكم.
- الرد على العملاء من المنصة وإرسال الردود تلقائيًا عبر الهاتف.

## المتطلبات

- Node.js
- GitHub (لرفع الكود)
- Render.com (لنشر المنصة مجانًا)
- Auto Forward SMS to PC / Phone على جوالك

## الملفات

1. `index.js` - كود السيرفر الرئيسي.
2. `public/index.html` - واجهة المستخدم.
3. `package.json` - إعدادات المشروع.

## كيفية التشغيل محليًا

1. انسخ المشروع على جهازك.
2. افتح Terminal.
3. نفذ:
```bash
npm install
npm start
```
4. افتح المتصفح على: `http://localhost:3000`

## النشر على Render

1. ارفع المشروع على GitHub.
2. سجل دخولك على [Render.com](https://render.com) وربط GitHub.
3. أنشئ Web Service جديد واختر المشروع.
4. Build Command: `npm install`
5. Start Command: `npm start`
6. احصل على الرابط الذي ستستخدمه كـ Webhook في تطبيق Auto Forward SMS.

## استخدام التطبيق مع Auto Forward SMS

- ضع رابط المنصة مع `/sms` في Webhook URL.
- اختر إرسال جميع الرسائل أو حسب الأرقام.
- الآن أي رسالة تصل للجوال تُرسل تلقائيًا للمنصة، ويستطيع الموظفون الرد مباشرة من الموقع.

## الملاحظات

- النسخة المجانية على Render قد تتوقف بعد فترة من عدم الاستخدام.
- يمكنك استخدام UptimeRobot للحفاظ على المنصة نشطة 24/7.
